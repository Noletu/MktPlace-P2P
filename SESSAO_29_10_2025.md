## 🎯 Objetivo Principal

Corrigir problema crítico de visibilidade em formulários KYC quando modo noturno (dark mode) estava ativado. Usuários reportaram que texto ficava invisível (branco em fundo branco).

---

## 🐛 Bug Reportado

### Descrição
Usuário relatou que no modo noturno, os campos de formulário do KYC ficavam invisíveis, impossibilitando o preenchimento.

### Evidência
- Screenshot fornecido pelo usuário
- Campos de texto brancos em fundo branco
- Labels invisíveis
- Helper texts ilegíveis

### Impacto
- 🔴 **CRÍTICO**: Funcionalidade KYC inutilizável em dark mode
- Usuários forçados a usar apenas light mode
- Experiência ruim em ambientes com pouca luz

---

## 🛠️ Correções Implementadas

### 1. KYC Level 2 Form
**Arquivo**: `apps/web/app/kyc/level2/page.tsx`

**Elementos Corrigidos**: 23

#### Labels (10 elementos)
Antes:
```tsx
<label className="block text-sm font-medium mb-2">CPF</label>
```

Depois:
```tsx
<label className="block text-sm font-medium mb-2 text-gray-900 dark:text-white">CPF</label>
```

**Campos corrigidos**:
- CPF
- Telefone
- Selfie com Documento
- Documento de Identidade
- CEP
- Rua
- Número
- Complemento
- Cidade
- Estado

#### Inputs/Selects (10 elementos)
Antes:
```tsx
<input className="w-full px-4 py-2 border rounded-lg focus:ring-2 focus:ring-blue-500" />
```

Depois:
```tsx
<input className="w-full px-4 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-white dark:bg-gray-800 text-gray-900 dark:text-white placeholder:text-gray-400 dark:placeholder:text-gray-500 focus:ring-2 focus:ring-blue-500 dark:focus:ring-blue-600 focus:border-transparent" />
```

#### Heading (1 elemento)
```tsx
<h3 className="font-semibold mb-4 text-gray-900 dark:text-white">Endereço</h3>
```

#### Helper Texts (2 elementos)
```tsx
<p className="text-xs text-gray-500 dark:text-gray-400 mt-1">
```

## 📐 Padrão de Dark Mode Estabelecido

### Labels
```tsx
className="text-gray-900 dark:text-white"
```

### Inputs/Selects/Textareas
```tsx
className="
  border-gray-300 dark:border-gray-600
  bg-white dark:bg-gray-800
  text-gray-900 dark:text-white
  placeholder:text-gray-400 dark:placeholder:text-gray-500
  focus:ring-blue-500 dark:focus:ring-blue-600
"
```
